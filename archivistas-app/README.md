# Como migrar a yarn

1.  Hacer pull al repositorio
2.  Asegurarse de que package.lock sea eliminado
3.  Ejecutar el comando "yarn"
4.  Ejecutar el comando "yarn -v" para asegurarse de que se haya instalado
